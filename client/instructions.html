📘 Cómo jugar

🎯 Objetivo: destruye la base enemiga mientras proteges la tuya.

🛡️ Mecánicas: construye torretas para defender y envía oleadas de enemigos para atacar. Mueve la cámara con el ratón o las flechas.

⚙️ Configuración:

  • 🗺️ Tipo de mapa: cambia el tamaño y forma del escenario.
  • 🏰 Cantidad de torres: cuántas torres iniciales tiene cada jugador.
  • 💰 Economía: velocidad con la que ganas dinero.
  • 🎨 Modo gráfico: elige entre estilo minimalista o con texturas.

¡Buena suerte y diviértete! 😄
